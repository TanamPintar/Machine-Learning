# Image Classification Model
|   Model | Classification Output |
| :----------------: | :----------------------------------------------------------------: |
| Transfer Learning : MobileNetV2 | Alluvial Soil |
| Input : Image(224, 224) | Black Soil |
| Output : 4 labels | Clay Soil |
| Total params: 10,286,788 | Red Soil |

| Time : 2454s |
